# PGV — Next.js Website

This is a modern website built with Next.js framework.

## Features

- Next.js 14 with App Router
- React components for all pages
- Dark/Light theme switching
- Responsive design
- Guestbook functionality (localStorage)
- Video gallery with YouTube embeds
- Admin authentication 
  
## Pages 

- `/` - Home page with hero section and recent articles
- `/clanky` - Articles page
- `/videa` - Videos page with YouTube embeds
- `/odkazy` - Useful links organized by categories
- `/vzkazy` - Guestbook page  

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Run linter
npm run lint
```

## Deployment

This project is configured for Vercel deployment. The `vercel.json` configuration file is set up for Next.js.

## Tech Stack

- Next.js 14
- React 18
- CSS with custom properties (CSS variables)
- Google Fonts (Inter)

## Original Project

Converted from a static HTML/CSS/JS website to Next.js framework.
